# 宝宝满月宴邀请页 


# 模块 / 功能
- 宝宝信息与暖心文案
- 满月宴倒计时
- 宴席时间与地点信息
- 添加到日历按钮 (ICS 下载，可导入系统日历)
- 地图导航按钮 (高德导航 iframe + App 跳转)
- 爸爸妈妈的 WhatsApp 联系按钮

# 技术栈
- Netlify (https://netlify.com)

- Bulma CSS (https://bulma.io)
