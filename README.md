# 简洁婚礼请柬落地页 


# 模块 / 功能
- 主要信息
- 婚礼倒计时
- 时间与地点信息
- 添加到日历按钮 (Google Calendar)
- 地图导航按钮 (Google Map)
- 发送消息按钮 (WhatsApp API)

# 技术栈
- Netlify (https://netlify.com)

- Bulma CSS (https://bulma.io)
